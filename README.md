# hybrid-apps
A dataset of 15,000 hybrid apps which use either:     
- Apache Cordova
- Appcelerator Titanium
- Adobe Air

## Import to your own mongodb instance
```
mongorestore --collection nameOfCollection --db nameOfDb pathToBsonFile
```
